# Protocols
- RS232
- RS485 (Max485)
  - master / slave connection
  - used in factories alot
  - for long distance
- I2C
  - PCF8574
  - https://www.youtube.com/watch?v=CAvawEcxoPU
  - not for distant communication, 100k~400kbps quite slow, for short distance and many devices
  - one master, many slaves
- SPI
